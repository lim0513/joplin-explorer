# Joplin Explorer

一个 [Joplin](https://joplinapp.org/) 插件，将笔记本和笔记合并显示在一个统一的树形侧边栏中，类似文件资源管理器。

[English](README.md)

## 功能

- **统一树形视图** — 笔记本和笔记显示在同一个可折叠面板中
- **自定义图标** — 显示笔记本设置中的 emoji 图标
- **收藏夹（Pin）** — 收藏笔记和笔记本，固定在树顶部可折叠区段；拖拽项目到收藏夹直接收藏，收藏夹内拖拽排序；重启后保持
- **增强搜索** — 同时搜索笔记、笔记本和标签，关键词高亮；结果按类型分组可折叠；支持子串匹配（如 "8121R" 可匹配 "KY8121R"）
- **排序** — 按修改时间或标题排序（升序/降序切换）
- **右键菜单**
  - 笔记本：收藏/取消收藏、新建笔记、新建待办、新建子笔记本、重命名、导出、删除
  - 笔记：收藏/取消收藏、打开、在新窗口打开、复制 Markdown 链接、复制副本、笔记/待办切换、切换完成状态、重命名、查看属性、删除
- **拖拽** — 拖拽笔记到其他笔记本，拖拽文件夹调整层级；拖到底部空白区创建新笔记本；拖到收藏夹直接收藏
- **同步按钮** — 一键触发同步，带状态反馈（同步中 → 同步完成）
- **自动展开** — 启动时自动展开到当前选中的笔记
- **全部折叠** — 一键折叠所有笔记本
- **滚动位置保持** — 切换笔记时不会跳到顶部
- **多语言** — 支持简体中文、繁体中文、英文、日语、俄语（跟随 Joplin 语言设置）

## 安装

### 从文件安装

1. 从 [最新 Release](https://github.com/lim0513/joplin-explorer/releases/latest) 下载 `joplin-explorer.jpl`
2. 打开 Joplin，进入 **工具 → 选项 → 插件**
3. 点击齿轮图标，选择 **从文件安装**
4. 选择下载的 `.jpl` 文件
5. 重启 Joplin

## 使用方法

安装后，Explorer 面板会出现在编辑器侧边。你可以：

- **点击** 笔记本展开/折叠
- **点击** 笔记打开它
- **右键** 弹出操作菜单（包含收藏/取消收藏）
- **拖拽** 笔记或笔记本来重新组织；拖到底部区域创建新笔记本；拖到收藏夹直接收藏
- 使用顶部 **工具栏** 快速新建笔记本/笔记/待办、排序、全部折叠
- 使用 **搜索栏** 搜索笔记、笔记本和标签
- 点击底部 **同步** 按钮触发同步

## 开发

```bash
# 安装依赖
npm install

# 构建
npm run dist

# 监听模式
npm run dev
```

构建产物输出到 `publish/` 目录。本地测试时，在 Joplin 设置中将 `plugins.devPluginPaths` 指向 `publish/` 目录即可。
