# Joplin Explorer — Roadmap

This document outlines the planned features and long‑term direction of **Joplin Explorer**, a unified tree‑view sidebar plugin for Joplin.  
The roadmap is divided into version milestones to provide clarity for users and contributors.

---

## ✅ Current Features
- Unified tree view (notebooks + notes)
- Emoji notebook icons
- Real‑time search (title filter)
- Sorting (updated time / title)
- Context menus for notebooks and notes
- Drag & drop reordering
- Sync button with status feedback
- Auto‑expand to current note
- Collapse All
- Scroll position retention
- i18n (EN / ZH‑CN / ZH‑TW)
